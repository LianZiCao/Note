## max ，min的初始化问题
```C++
int max = -INT_MAX;
int min = INT_MAX;
```
当然也可以令其等于比较对象中的某个数

## 比较结果总是是最后一个数
忘记更新min/max，例如
```c++
if (t > max)
{
    a = i;
    b = j;
}
```
**错了**，代码块中应该加一行`max = t;`

## 误用=表达式的情况
（来源于前缀和与差分.md的地毯例题）
```c++
pre[x1][y1] = pre[x2 + 1][y2 + 1] += 1;
pre[x2 + 1][y1] = pre[x1][y2 + 1] += -1;
```
应该改为
```c++
pre[x1][y1] += 1;
pre[x2 + 1][y2 + 1] += 1;
pre[x2 + 1][y1] += -1;
pre[x1][y2 + 1] += -1;
```
因为`pre[x1][y1]`和`pre[x2 + 1][y2 + 1]`的值不一定始终一致
